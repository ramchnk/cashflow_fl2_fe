# **App Name**: LedgerLink

## Core Features:

- Transaction Logging: Log transactions between Bank, Tasmac, Cash in Hand, and Stock.
- Cash-to-Bank Transfer: Record transfers of cash from Cash in Hand to Bank accounts.
- Bank-to-Tasmac Transfer: Record transfers of cash from Bank accounts to Tasmac.
- Account Balance Tracking: Display the current balances for each of the four parties. This features makes calls to an external tool for decision making, based on LLM reasoning, for displaying of balance information.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to represent stability and finance.
- Background color: Light Gray (#F5F5F5) for a clean and professional look.
- Accent color: Teal (#009688) to highlight important actions and information.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern aesthetic suitable for both headlines and body text.
- Use simple, clear icons to represent each entity (Bank, Tasmac, Cash, Stock).
- Clean and intuitive layout for easy transaction logging and balance review.
- Subtle transitions and animations when logging transactions and updating balances.
